# Ясное письмо на русском

Принципы Норы Галь («Слово живое и мёртвое») и Максима Ильяхова («Пиши, сокращай»).

## Краткая памятка

| Заменяй | На |
|---------|-----|
| Пассивный залог | Активный залог |
| Отрицания | Утверждения |
| Абстракции | Конкретные числа/факты |
| Отглагольные (-ание, -ение) | Глаголы |
| Слова-паразиты | Ничего |
| Канцеляризмы | Живой язык |

## Порядок работы

1. Напиши или получи черновик
2. Примени правила из [references/style-guide.md](references/style-guide.md)
3. Проверь по чек-листу в конце того файла
4. Убедись, что нет AI-штампов

## Примеры

**Было:**
> В рамках данного проекта была осуществлена разработка системы, которая обеспечивает высокую скорость обработки запросов.

**Стало:**
> Мы разработали систему, которая отвечает за 50 мс.

**Было:**
> Следует отметить, что данное решение имеет ключевое значение для пользователей.

**Стало:**
> Это решение сокращает время ожидания втрое.

## Главные правила

**Активный залог:** «Сервер обрабатывает», а не «обрабатывается сервером»

**Утверждения:** «Забыл», а не «не помнил»

**Конкретика:** «12 000 пользователей», а не «много пользователей»

**Глаголы:** «Проверить», а не «осуществить проверку»

**Без воды:** Удаляй «следует отметить», «ни для кого не секрет», «в целом»

**Без канцелярита:** «Сейчас», а не «на сегодняшний день»

**Без AI-штампов:** Избегай «играет ключевую роль», «уникальный», «инновационный»
